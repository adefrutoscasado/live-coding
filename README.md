
### Installation

Requires [Node.js](https://nodejs.org/) v11.13+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd live-coding
$ npm install
$ npm start
```

## Available scripts

Build the app

```sh
$ npm run tsc
```

Execute the tests

```sh
$ npm run test
```

Execute the scraper

```sh
$ npm run scrap
```
